# MarketOnline– React

Aplicación web desarrollada con **React** que implementa una tienda de productos con funcionalidades de **administración básica**.  
El proyecto está orientado a demostrar la **correcta separación de componentes**, el **manejo de estado** y la reutilización de lógica en un entorno frontend moderno.

Este proyecto fue desarrollado en paralelo a una versión en Vue.js con el objetivo de **comparar enfoques y patrones** entre ambos frameworks.

---

## Tecnologías Utilizadas

- React
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router
- Hooks (useState, useEffect)

---

## Funcionalidades

- Listado de productos
- Creación de productos
- Edición de productos
- Eliminación de productos
- Componentes reutilizables
- Flujo de datos controlado mediante estado
